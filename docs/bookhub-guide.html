BookHub – Developer & Admin Guide

Overview of the custom WordPress theme, Book post type plugin, file responsibilities, and how they connect.

1. What’s included

  • wp-content/plugins/bookhub-library/bookhub-library.php: Registers Book post type and taxonomies (Genre, Author), adds meta fields (Year, Rating, ISBN), and a simple block.
  • wp-content/themes/bookhub/functions.php: Enables title tags, thumbnails, Tailwind CDN, registers the Primary Menu, and the Primary Sidebar.
  • wp-content/themes/bookhub/header.php / footer.php: Responsive layout, mobile menu, site search.
  • wp-content/themes/bookhub/front-page.php: Homepage hero, Top Rated Books, Latest Arrivals.
  • wp-content/themes/bookhub/home.php: Posts index when using “Your latest posts”.
  • wp-content/themes/bookhub/index.php: Fallback listing for posts.
  • wp-content/themes/bookhub/archive-book.php: Book archive with filters and grid cards.
  • wp-content/themes/bookhub/single-book.php: Book details with related books by Genre.
  • wp-content/themes/bookhub/taxonomy-genre.php / taxonomy-author.php: Taxonomy listings.
  • wp-content/themes/bookhub/search.php / searchform.php: Site-wide search and form styles.

WordPress resolves templates per request via its Template Hierarchy, then includes header.php and footer.php. The plugin exposes content types that the theme renders.

3. Setup checklist

  1. Activate plugin: BookHub Library (adds Books, Genres, Authors).
  2. Menus: Create a menu and assign to Primary Menu.
  3. Homepage: Either set a page as static homepage (uses front-page.php) or keep “Your latest posts” (uses home.php).
  4. Add Books with cover images, rating, year, ISBN, and assign Genres/Authors.
  5. Optional: Add widgets to Primary Sidebar.

4. Customization points

  • Branding: In header.php, update the logo badge and site title. Tailwind brand color is configured inline in functions.php via wp_add_inline_script('tailwind-cdn', ...).
  • Cards/Layouts: Grid/card styles are Tailwind utility classes; adjust directly in templates.
  • Queries: Featured and Latest sections in front-page.php use WP_Query. Modify posts_per_page, sorting, or add filters.
  • Related Books: Logic in single-book.php uses shared Genres. Change to Authors or Rating as needed.
  • Search: search.php renders mixed results. Limit to books by passing post_type=book in URLs or customizing the query.
